Mods

Included below is a list of all the mods the server is using. You can check out the modpack at modrinth!! In order to install this modpack you can use the modrinth launcher or a launcher such as MultiMc. If you want to suggest mods to be added to the server then join the discord. :D

Simple Voice Chat
Adds voice chat features including proximity chat.

Dragon Drops Elytra.
Ender Dragon now drops an Elytra.

Creeper Overhaul
Adds biome specific creepers.

Friends & Foes
Adds previously cut minecraft content. Mostly from mob vote losers.

Tool Trims
Adds customization to you tools in the form of tool trims.
